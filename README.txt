To Run:
1. Download as a zip file, extract as normal
2. Install Python 3.11
3. Download PySide6 (this should be the only library required, if not download PyQt6 as well)
4. Run main.py

IDE: PyCharm Community Edition 2023.2.1
NOTE: YOU DO NOT NEED AN IDE TO RUN THIS, IT CAN BE RUN ON TERMINAL BY CHANGING DIRECTORY IF YOU DO NOT WANT TO INSTALL AN IDE
